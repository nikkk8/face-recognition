# Face Recognition System using OpenCV 👤

A real-time face recognition system developed using Python and OpenCV. The project includes both standard face recognition and an enhanced recognition system capable of identifying unknown faces with an alert sound notification.

---

## 🚀 Features

- Real-time face detection and recognition
- Recognition using stored face image dataset
- Enhanced detection for unknown faces
- Automatic "Unknown" face labeling
- Beep sound alert for unauthorized/unknown individuals
- Webcam-based live recognition

---

## 🛠️ Technologies Used

- Python
- OpenCV
- NumPy
- Face Recognition
- Computer Vision
